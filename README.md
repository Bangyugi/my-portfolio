# Personal IT Portfolio

A fully responsive, modern personal IT portfolio website built with ReactJS, Vite, TailwindCSS, and Framer Motion.

## Features

- **Responsive Design**: Mobile-first approach ensuring great experience on all devices.
- **Modern UI/UX**: Minimalist dark theme with smooth animations.
- **Sections**: Hero, About, Skills, Projects, Experience, Contact.
- **Performance**: Optimized with Vite and lazy loading ready.

## Tech Stack

- **Framework**: ReactJS + Vite
- **Styling**: TailwindCSS
- **Routing**: React Router DOM (Setup for future expansion)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/my-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

## Deployment

This project can be easily deployed to Vercel, Netlify, or GitHub Pages.

### Vercel (Recommended)

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root.
